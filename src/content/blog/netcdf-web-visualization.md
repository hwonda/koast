## NetCDF란

NetCDF(Network Common Data Form)는 기상, 해양, 대기 과학 분야에서 널리 사용되는 데이터 포맷입니다. 다차원 배열 형태의 과학데이터를 효율적으로 저장하고 공유할 수 있도록 설계되었습니다.

## 웹 시각화의 과제

NetCDF 파일은 바이너리 포맷이며, 파일 크기가 수백 MB에서 수 GB에 이를 수 있습니다. 이를 웹 브라우저에서 직접 처리하기에는 여러 제약이 있습니다.

- 파일 크기가 커서 네트워크 전송에 부담
- 바이너리 포맷이라 클라이언트 직접 파싱이 어려움
- 다차원 데이터를 2D 화면에 표현하는 방법 필요

## 접근 방법

### 1. 서버 사이드 처리

서버에서 NetCDF를 읽어 JSON, 이미지 타일, 또는 GeoJSON으로 변환하여 클라이언트에 전달합니다.

- Python (xarray, netCDF4)
- Java (NetCDF-Java)
- OGC 표준 서비스 (WMS, WCS)

### 2. 클라이언트 렌더링

변환된 데이터를 브라우저에서 지도나 그래프로 표출합니다.

- Leaflet, OpenLayers (2D 지도)
- CesiumJS (3D 지구)
- D3.js, Chart.js (그래프)
- WebGL 기반 커스텀 렌더링

### 3. 타일링 전략

대용량 격자 데이터는 지도 타일 형태로 미리 생성하여 제공하면 성능을 확보할 수 있습니다.

## 한국해양기상기술의 접근

한국해양기상기술은 자체 개발한 과학데이터 가시화 엔진을 통해 NetCDF, GRIB2 등 다양한 포맷의 데이터를 웹에서 실시간으로 시각화할 수 있는 시스템을 구축합니다.

격자 데이터, 벡터 데이터, 시계열 데이터를 지도 위에 레이어로 표출하고, 시간에 따른 애니메이션도 지원합니다.

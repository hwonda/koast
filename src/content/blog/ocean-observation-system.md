## 해양기상 관측 시스템이란

해양기상 관측 시스템은 바다와 대기의 상태를 실시간으로 측정하고, 수집된 데이터를 전송·저장·표출하는 일련의 기술 체계입니다.

기상청, 해양 관련 기관, 연구기관 등에서 해양 안전, 재난 대응, 기후 연구 등을 위해 운용합니다.

## 시스템 구성요소

### 1. 센서

현장에서 물리적 변화를 감지하는 장치입니다.

- 풍향·풍속계
- 기압계
- 온·습도계
- 파고계
- 조위계
- 해수온도계

### 2. 데이터로거

센서에서 수집된 아날로그/디지털 신호를 일정 주기로 기록하고 저장합니다.

- 다채널 동시 수집
- 현장 저장 및 전송
- 원격 설정 변경 지원

### 3. 통신장비

수집된 데이터를 원격지 서버로 전송합니다.

- CDMA/LTE/M2M 통신
- 위성통신 (해상 원거리)
- 이더넷/시리얼 통신

### 4. 수집 서버

전송된 데이터를 수신하고 데이터베이스에 저장합니다.

- 실시간 데이터 수신
- 품질관리(QC) 처리
- 이상값 탐지 및 보정

### 5. 표출 시스템

저장된 데이터를 사용자가 이해할 수 있는 형태로 시각화합니다.

- 웹 기반 대시보드
- 지도 위 관측소별 데이터 표출
- 시계열 그래프
- 실시간 모니터링 화면

## 데이터 흐름

```
센서 → 데이터로거 → 통신장비 → 수집서버 → DB → 표출시스템
```

각 단계에서 데이터의 신뢰성을 확보하는 것이 관측 시스템의 핵심입니다.

## 마무리

해양기상 관측 시스템은 단순히 센서를 설치하는 것이 아니라, 데이터의 수집부터 표출까지 전 과정을 설계해야 합니다. 한국해양기상기술은 이 전 과정을 통합적으로 구축할 수 있는 기술력을 보유하고 있습니다.
